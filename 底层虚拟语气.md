# 底层虚拟语气

> 相关：[底层时态.md](./底层时态.md) · [词类.md](./词类.md) · [底层索引.md](./底层索引.md)

---

## 要解决的问题

**「说的不是事实，是假设、愿望、建议、与现在/过去相反的情况。」**

- 如果我是你……（我不是你）  
- 我希望他现在在这儿……（他不在）  
- 要是我昨天去了就好了……（我没去）  

---

## 底层逻辑

| 说的事实吗 | 动词形式 |
|-----------|---------|
| 是事实 | 正常时态 |
| 假设/相反/愿望 | **虚拟形式**（往往用过去式表示「非真实」） |

---

## 常见场景

### 1. 与现在事实相反

| 中文 | 英文 | 结构 |
|------|------|------|
| 如果我有时间（其实没有） | If I **had** time, I would go. | If + 过去式, would + 动词 |
| 要是我像你一样（我不是） | If I **were** you, I would … | were（虚拟常用） |

### 2. 与过去事实相反

| 中文 | 英文 |
|------|------|
| 要是我昨天去了（没去） | If I **had gone**, I **would have seen** him. |
| | If + had done, would have done |

### 3. 愿望

| 中文 | 英文 |
|------|------|
| 我希望现在能飞 | I wish I **could** fly. |
| 真希望昨天没说那个 | I wish I **hadn't said** that. |

### 4. 建议/要求（that 从句）

| 中文 | 英文 |
|------|------|
| 我建议他走 | I suggest he **go** / **should go**. |
| 重要的是他来 | It's important that he **be** on time. |

（正式语法用动词原形，口语常用 should）

---

## 实操流程

```
这是真的吗？
    ↓ 否
假设/愿望/建议 → 选虚拟结构
    ↓
现在相反？→ If + 过去, would ...
过去相反？→ If + had done, would have done
```

---

## 标签

**虚拟语气（Subjunctive Mood）**

---

## 一句话

**虚拟语气 = 标「这不是事实」，用特殊动词形式表达假设、愿望、与 reality 相反的情况。**
