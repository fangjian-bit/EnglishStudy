# 底层固定搭配

> 相关：[底层短语动词.md](./底层短语动词.md) · [词类.md](./词类.md) · [底层索引.md](./底层索引.md)

---

## 要解决的问题

**「语法没错，但英语不这么说。」**

- ~~do a decision~~ → **make** a decision  
- ~~strong rain~~ → **heavy** rain  
- ~~open the light~~ → **turn on** the light  

**搭配 = 哪些词习惯上绑在一起说**，不是规则能完全推出来的。

---

## 底层逻辑

| 层面 | 能推吗 | 例子 |
|------|--------|------|
| 语法结构 | 能 | 主语 + 动词 + 宾语 |
| 词与词的搭配 | 往往不能 | make vs do, heavy vs strong |

母语者靠 **听得多了**，学习者靠 **记常见块（chunk）**。

---

## 常见类型

### 1. 动词 + 名词

| 搭配 | 意思 | 错例 |
|------|------|------|
| make a decision | 做决定 | ~~do a decision~~ |
| make a mistake | 犯错 | |
| do homework | 做作业 | ~~make homework~~ |
| take a shower | 洗澡 | ~~have a shower~~（英式也可） |
| have breakfast | 吃早餐 | |

### 2. 形容词 + 名词

| 搭配 | 错例 |
|------|------|
| **heavy** rain | ~~strong rain~~ |
| **strong** coffee | |
| **fast** food | |
| **deep** sleep | |

### 3. 动词 + 介词/副词（与短语动词重叠）

| 搭配 | 意思 |
|------|------|
| depend **on** | 依赖 |
| listen **to** | 听 |
| good **at** | 擅长 |
| interested **in** | 对…感兴趣 |

### 4. 固定短语

| 短语 | 意思 |
|------|------|
| by the way | 顺便说 |
| at least | 至少 |
| as soon as | 一…就… |
| look forward to | 期待（to 是介词，后接 -ing） |

---

## 学习法（底层思维）

1. **整块记**，不要逐词翻译  
2. 遇到新搭配问：**「英语习惯用哪个动词/形容词配这个名词？」**  
3. 读/听时留意：**哪几个词总一起出现**  
4. 和语法分开：**语法对 ≠ 地道**

---

## 实操

| 中文 | 先想「英语常和什么绑一起」 | 英文 |
|------|--------------------------|------|
| 做决定 | make + decision | make a decision |
| 下大雨 | heavy + rain | heavy rain |
| 对…感兴趣 | interested in | be interested in … |
