# 03 · 宾补与不定式 · 例句解析

> 理论 → [../底层笔记/底层英语.md](../底层笔记/底层英语.md)

---

## 1

**英文：** Keep the room clean.  
**中文：** 保持房间整洁。

| 成分 | 词 |
|------|-----|
| 谓语 | Keep |
| 宾语 | the room |
| 宾补 | clean |
| 结构 | **动词 + 宾语 + 形容词宾补** |

**意图：** 动作之后，房间 **怎么样**？→ clean。心里换算：the room **is** clean。

---

## 2

**英文：** They elected him president.  
**中文：** 他们选他当总统。

| 成分 | 词 |
|------|-----|
| 主语 | They |
| 谓语 | elected |
| 宾语 | him |
| 宾补 | president |
| 结构 | **动词 + 宾语 + 名词宾补** |

**意图：** 选完之后，他 **是什么身份**？→ president。心里换算：he **became** president。

**注意：** Tom 是宾补，不是第二个宾语——说明 him **叫什么**。

## 4

| 成分 | 词 |

| 主语 | I |
| 谓语 | find |
| 宾语 | it |
| 宾补 | interesting |

## 5

**英文：** Let me go.  
**中文：** 让我走。

| 成分 | 词 |
|------|-----|
| 谓语 | Let |
| 宾语 | me |
| 宾补 | go |
| 结构 | **框 B：let + 宾语 + 动词原形** |

**意图：** 直接 **放行**，不是对「走这件事」表态 → **不加 to**。

## 6

**英文：** Allow them to leave.  
**中文：** 允许他们离开。

| 成分 | 词 |
|------|-----|
| 谓语 | Allow |
| 宾语 | them |
| 宾补 | to leave |
| 结构 | **框 A：allow + 宾语 + to do** |

**意图：** 对「离开这件事」 **给许可** → to 标出被允许的内容。❌ Allow them leave。

## 7

**英文：** Let them leave.  
**中文：** 让他们走 / 不拦他们走了。

| 成分 | 词 |
|------|-----|

| 宾语 | them |
| 宾补 | leave |
| 结构 | **框 B：let + 宾语 + 动词原形** |

## 33

**英文：** He is too young to drive.  
**中文：** 他太小，不能开车。

| 成分 | 词 |
|------|-----|
| 主语 | He |
| 谓语 | is |
| 表语 | too young |
| 状语 | to drive |
| 结构 | **主系表 + 结果状语（不定式）** |

**注意：** to drive 不是宾补——没有「中间宾语」。too…to… = 太……而不能……，不定式表结果。
