# 底层语篇衔接

> 相关：[底层语用.md](./底层语用.md) · [底层英语.md](./底层英语.md) · [底层索引.md](./底层索引.md)

---

## 要解决的问题

**「一个句子说完了，下一句怎么接？一段里想法怎么连？」**

单句语法对，拼成段落可能 still 乱 —— 需要 **衔接词** 和 **逻辑关系**。

---

## 底层逻辑

| 问题 | 解法 |
|------|------|
| 接着上面说 | 并列、递进 |
| 转话题 | 转折 |
| 解释原因 | 因果 |
| 举例 | 举例 |
| 总结 | 总结 |
| 顺序 | 时间/步骤 |

---

## 常见衔接词（按关系）

### 并列 / 递进

| 词 | 作用 | 例子 |
|----|------|------|
| and, also, besides | 还有 | I like tea, **and** I like coffee. |
| moreover, furthermore, in addition | 而且（正式） | **Moreover**, … |
| not only … but also | 不仅…而且 | |

### 转折

| 词 | 例子 |
|----|------|
| but, however, yet | I tried; **however**, I failed. |
| although / though | **Although** it rained, we went. |
| on the other hand | 另一方面 |

### 因果

| 词 | 例子 |
|----|------|
| because, since, as | **Because** it rained, we stayed. |
| so, therefore, thus | It rained; **therefore**, we stayed. |
| as a result | 结果 |

### 举例

| 词 | 例子 |
|----|------|
| for example, for instance | **For example**, … |
| such as | fruits **such as** apples |

### 顺序 / 步骤

| 词 | 例子 |
|----|------|
| first, second, finally | **First**, … **Then**, … |
| before, after, when | **After** that, … |

### 总结

| 词 | 例子 |
|----|------|
| in conclusion, to sum up | **In conclusion**, … |
| in short, overall | |

---

## 段落结构（写作底层）

```
主题句（这段说什么）
    ↓
展开（原因、例子、细节）
    ↓
衔接词连接各句
    ↓
可选：小结句
```

---

## 语篇 vs 句子

| | 句子层面 | 语篇层面 |
|--|---------|---------|
| 关注 | 主谓宾对不对 | 前后逻辑顺不顺 |
| 工具 | 语法 | 衔接词、段落结构 |

---

## 学习法

1. 读文章 **标衔接词** —— 看作者怎么连想法  
2. 写作前先列 **逻辑链**：因为A所以B，但是C  
3. 每段开头问：**这段和上一段什么关系？**  

---

## 标签

**语篇（Discourse）** · **衔接（Cohesion）** · **连接词（Connector/Transition）**

---

## 一句话

**语篇衔接 = 句与句、段与段之间用逻辑关系和衔接词连起来，让听者/读者跟得上思路。**
