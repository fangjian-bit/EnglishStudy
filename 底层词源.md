# 底层词源

> **很多英语词不是随机字母，而是「古义 + 古义」拼出来的。**  
> 搞懂词源 = 见到生词能猜大概方向，记一个词带出一串词。  
> 相关：[底层构词法.md](./底层构词法.md) · [词类.md](./词类.md) · [单词积累.md](./单词积累.md) · [底层索引.md](./底层索引.md)

## 怎么用这份笔记

1. **不要只背中文意思** —— 问：这个词的古义是什么？和哪个词根/前缀拼的？
2. **记「词族」** —— 同一个根的一串词一起记（forward / forecast / before…）
3. **猜完要查** —— 词源帮助理解，不保证 100% 对

**推荐顺序：**

```
看到一个词
    ↓
能拆开吗？（前缀 + 词根 + 后缀）
    ↓
各部分古义是什么？
    ↓
合起来推大概意思 → 查词典确认
```

## 1. 英语词从哪来（底层背景）

现代英语 ≈ **日耳曼语（古英语）** + **拉丁/法语** + 少量希腊语。

| 来源 | 感觉 | 例子 |
|------|------|------|
| 日耳曼/古英语 | 日常、短、口语感强 | for, hand, water, forward |
| 拉丁/法语 | 正式、学术、长 | predict, transport, visible |
| 希腊 | 科学、医学、技术 | biology, telephone, psychology |

**一句话：** 同一个意思，英语里常有两套词 —— 短的往往更日耳曼，长的往往更拉丁/法语。

## 2. for- / fore- / before：「在前、向前」

你举的例子就在这一类。

### for / fore 的古义

| 古义 | 现代残留 |
|------|---------|
| **在前、向前** | for**ward**, **fore**head, **fore**arm |
| **事先、提前** | **fore**cast, **fore**see, **fore**sight |
| **前面那部分** | **fore**word（书的前言，注意拼写 fore-） |

### forward 怎么来的

```
for（向前）+ -ward（方向）
    ↓
forward = 朝前的、向前
```

同类 **-ward / -wards** = 「朝…方向」：

| 词 | 拆 | 意思 |
|----|-----|------|
| forward | for + ward | 向前 |
| backward | back + ward | 向后 |
| upward | up + ward | 向上 |
| downward | down + ward | 向下 |
| inward | in + ward | 向内 |
| outward | out + ward | 向外 |
| toward(s) | to + ward | 朝向 |
| afterward(s) | after + ward | 之后 |
| homeward | home + ward | 朝家 |

### before 也是「在前」

```
be（by，在）+ fore（前）
    ↓
before = 在…之前
```

### fore- 词族（记一串）

| 词 | 词源思路 | 意思 |
|----|---------|------|
| forehead | fore + head | 前额（头前面） |
| forearm | fore + arm | 前臂 |
| foreground | fore + ground | 前景 |
| forefront | fore + front | 最前沿 |
| foresee | fore + see | 预见 |
| foretell | fore + tell | 预言 |
| forecast | fore + cast | 预报（提前「扔」出判断） |
| forewarn | fore + warn | 预先警告 |
| beforehand | before + hand | 事先 |

### ⚠️ for- 不总是「向前」

古英语里 **for-** 有时还表示 **完全、彻底、加强**（现在多体现在少数词里）：

| 词 | 古义思路 | 现代意思 |
|----|---------|---------|
| forgive | for（完全）+ give | 原谅（完全给出？古义演变） |
| forget | for（完全）+ get | 忘记 |
| forbid | for（against）+ bid（命令） | 禁止（命令不许） |

**底层结论：** 同一个前缀，古义可能分化 —— **forward 的 for 是「向前」，forbid 的 for 是「反对」**。拆词时要查具体词，不能一条规则硬套。

## 8. 易混：形近但词源不同

| 词 A | 词 B | 差在哪 |
|------|------|--------|
| **fore**word（前言） | **for**ward（向前） | fore=前部；for=向前 |
| respect | inspect | re- vs in- + spect（看） |
| immigrate | emigrate | im=进 vs e=出 + migrate（迁移） |
| historic | historical | historic=历史上重要的；historical=与历史有关的 |
| later | latter | later=更晚；latter=后者 |

---

## 9. 学习法（底层思维）

1. **记词族，不记孤词** —— 见到 forward，把 forecast / foresee / before 一起扫一遍  
2. **先拆再背** —— transport = trans + port，比死记「运输」牢  
3. **区分日耳曼短词 vs 拉丁长词** —— 同义时往往风格不同（buy vs purchase）  
4. **词源是辅助，不是替代词典** —— 古义演变复杂，猜完要确认

## 12. 一句话总结

> **很多英语词 = 古义 A + 古义 B 拼起来；搞懂 for 为什么能生出 forward，就搞懂了词源学习的意义——不是背更多中文，而是看见词内部的逻辑。**
