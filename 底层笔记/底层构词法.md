# 底层构词法

> 相关：[底层词源.md](./底层词源.md) · [单词积累.md](./单词积累.md) · [索引.md](./索引.md)

---

## 要解决的问题

**「见到生词，能不能从零件猜出大概意思？」**

- unhappy = un + happy  
- teacher = teach + er  
- impossible = im + possible  

**构词法 = 词是怎么拼出来的。**

---

## 三类零件

| 零件 | 作用 | 例子 |
|------|------|------|
| **词根（root）** | 核心意思 | act, happy, possible |
| **前缀（prefix）** | 加在词根前，改意思 | un-, re-, pre-, im- |
| **后缀（suffix）** | 加在词根后，常改词性 | -er, -tion, -ly, -ness |

---

## 常见前缀（改意思）

| 前缀 | 大概意思 | 例子 |
|------|---------|------|
| un- / im- / in- / dis- | 不、相反 | **un**happy, **im**possible, **dis**agree |
| re- | 再、回 | **re**write, **re**turn |
| pre- | 前、预先 | **pre**view, **pre**fix |
| mis- | 错 | **mis**take |
| over- | 过度 | **over**work |
| under- | 不足、在…下 | **under**stand |

---

## 常见后缀（常改词性）

| 后缀 | 常变成 | 例子 |
|------|--------|------|
| -er / -or | 名词（人/物） | teach → teach**er** |
| -tion / -sion | 名词（行为/结果） | act → act**ion** |
| -ment | 名词 | develop → develop**ment** |
| -ness | 名词（性质） | happy → happy**ness** |
| -ly | 副词 | quick → quick**ly** |
| -ful / -less | 形容词 | care → care**ful** / care**less** |
| -able / -ible | 形容词（能…的） | read → read**able** |

---

## 合成词（两个词合成一个）

| 类型 | 例子 |
|------|------|
| 名词+名词 | **class**room, **foot**ball |
| 形容词+名词 | **black**board |
| 动词+副词 | **break**down（也是短语动词） |

---

## 学习法

1. 遇到生词 **拆零件**  
2. 认常见前缀后缀 → **猜词性、猜大概意思**  
3. 猜完 **查词典确认** —— 构词不是 100% 准确

## 一句话

**构词法 = 词由「词根+前缀+后缀」拼成，帮助猜词义和词性，但不能代替查词典。**
