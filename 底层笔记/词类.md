# 词类

> **词类不是「这个词属于哪一类所以要这样用」，而是「人要表达某种信息，需要这类词来承担某种角色」。**  
> 底层逻辑见 → [底层英语.md](./底层英语.md)  
> 具体词汇见 → [单词积累.md](./单词积累.md)  
> **全部专题索引**见 → [索引.md](./索引.md)

---

## 怎么用这份笔记

和 [底层英语.md](./底层英语.md) 一样，每个词类只问三个问题：

1. **没有这类词，会表达不了什么？**
2. **它在句子里通常承担什么角色？**
3. **能不能用一个自己的例子？**

**推荐顺序：**

```
我想表达什么信息？
    ↓
这类信息需要哪种词来承载？
    ↓
英语里这类词叫什么？（标签，放最后）
```

---

## 1. 总览：词分两大类

| 大类 | 底层区别 | 包含 |
|------|---------|------|
| **实词（内容词）** | 承载**具体意思**：人、事、物、动作、性质 | 名词、动词、形容词、副词、数词 |
| **虚词（功能词）** | 不单独表意，负责**连接、限定、辅助** | 代词、冠词/限定词、介词、连词、助动词、感叹词 |

**一句话：** 实词是「说什么」，虚词是「把实词组织起来」。

---

## 2. 名词（Noun）

### 要解决的问题

「说的是**谁 / 什么 / 哪种东西 / 哪种概念**？」

### 它在句子里干什么

| 角色 | 例子 | 句子成分 |
|------|------|---------|
| 作锚点 | **The dog** barks. | 主语 |
| 作动作对象 | I eat **an apple**. | 宾语 |
| 作描述 | She is **a teacher**. | 表语 |
| 被修饰 | a **red** car | 被定语修饰 |

### 底层理解

名词 = **命名**：给世界上的人、物、地点、想法、抽象概念贴标签。

- 具体：dog, apple, Beijing
- 抽象：love, time, idea

### 标签

**名词（Noun）** — 语法书缩写 n.

---

## 3. 代词（Pronoun）

### 要解决的问题

「已经知道说的是谁/什么，**不想重复说名字**。」

### 它在句子里干什么

**代替名词**，承担名词能承担的角色（主语、宾语等）。

| 类型 | 解决什么问题 | 例子 |
|------|-------------|------|
| 人称代词 | 代替具体的人/物 | **He** is happy. / I like **it**. |
| 物主代词 | 代替「某人的某物」 | **My** book / The book is **mine**. |
| 指示代词 | 指「这个/那个」 | **This** is good. |
| 不定代词 | 指不确定的人/物 | **Someone** came. / **Nothing** happened. |
| 疑问代词 | 提问「谁/什么」 | **Who** are you? / **What** is this? |
| 关系代词 | 连接定语从句 | The man **who** called you … |
| 反身代词 | 动作回到自身 | He hurt **himself**. |

### 底层理解

代词 = **名词的替身**，避免重复。

### 标签

**代词（Pronoun）** — 缩写 pron.

常见词见 → [单词积累.md](./单词积累.md)

---

## 4. 动词（Verb）

### 要解决的问题

「这个主语**怎么了 / 做了什么 / 处于什么状态**？」

### 它在句子里干什么

| 角色 | 例子 |
|------|------|
| 作谓语核心 | He **runs**. / She **is** happy. |
| 决定后面要不要宾语/表语 | eat → 要宾语；sleep → 不要；is → 要表语 |

### 动词的大分类（按「意思」分，不是死背）

| 类型 | 解决什么问题 | 例子 |
|------|-------------|------|
| **动作动词** | 描述一个动作 | run, eat, write |
| **状态动词** | 描述持续状态 | know, believe, love |
| **系动词** | 连接主语和描述（几乎无动作） | be, look, feel, become |
| **及物动词** | 意思指向对象 | like, eat → 要宾语 |
| **不及物动词** | 意思不指向对象 | sleep, arrive |
| **兼类动词** | 两种都行 | eat / eat rice |

### 底层理解

动词 = 句子的**发动机**：没有动词，只有名词，像贴标签，没有「事件」或「状态变化」。

### 标签

**动词（Verb）** — 缩写 v.

及物/不及物常见词见 → [单词积累.md](./单词积累.md)

## 5. 助动词（Auxiliary Verb / Helping Verb）

### 要解决的问题

**单独一个动词表达不了**的信息：时态、进行、被动、疑问、否定、语气等。

### 它在句子里干什么

**辅助主要动词**，本身通常不是句意核心，但承担语法功能。

| 助动词 | 解决什么问题 | 例子 |
|--------|-------------|------|
| **be** | 进行时 / 被动语态 | I **am** working. / It **was** broken. |
| **have** | 完成时 | I **have** finished. |
| **do** | 疑问 / 否定 / 强调 | **Do** you like it? / I **don't** know. / I **do** like it! |

### 底层理解

助动词 = **给主要动词加层语法信息**：时间、状态、语气、句式。

没有助动词，很多英文句式说不成：
- ❌ You like it? → ✅ **Do** you like it?
- ❌ I not know → ✅ I **do** not know

### 标签

**助动词（Auxiliary Verb）** — 缩写 aux.

---

## 6. 情态动词（Modal Verb）

### 要解决的问题

「是**能力 / 许可 / 义务 / 可能 / 必要**中的哪一种？」

### 它在句子里干什么

**情态动词是一种特殊的助动词**，后面接动词原形，表达说话人的态度。

| 情态动词 | 解决什么问题 | 例子 |
|---------|-------------|------|
| can / could | 能力 / 许可 | I **can** swim. / **Can** I go? |
| may / might | 可能 / 许可 | It **might** rain. |
| must | 必须 / 推测 | You **must** go. |
| should / ought to | 应该 | You **should** rest. |
| will / would | 意愿 / 将来 / 礼貌 | I **will** help. / **Would** you mind? |
| shall | 正式提议 / 将来（英式） | **Shall** we go? |

### 底层理解

情态动词 = **在事实外面加一层「态度」**：能不能、要不要、会不会、该不该。

### 标签

**情态动词（Modal Verb）** — 属于助动词的一种。

---

## 7. 形容词（Adjective）

### 要解决的问题

「是**哪一种 / 什么样的**？」—— 名词太粗，要细分。

### 它在句子里干什么

| 角色 | 例子 | 句子成分 |
|------|------|---------|
| 修饰名词 | a **red** car | **定语** |
| 作表语（描述主语） | She is **happy**. | **表语** |

### 底层理解

形容词 = **给名词加细节**：大小、颜色、好坏、数量感（many 等限定词有时单列）。

### 标签

**形容词（Adjective）** — 缩写 adj.

---

## 8. 副词（Adverb）

### 要解决的问题

「**怎样 / 何时 / 何地 / 多少 / 多频繁**？」—— 动词或整句需要细化。

### 它在句子里干什么

| 修饰对象 | 例子 | 句子成分 |
|---------|------|---------|
| 动词 | runs **quickly** | **状语** |
| 形容词 | **very** happy | 修饰形容词 |
| 副词 | **quite** slowly | 修饰副词 |
| 整句 | **Unfortunately**, he left. | **状语** |

### 常见副词类型

| 类型 | 回答什么 | 例子 |
|------|---------|------|
| 方式 | 怎样 | quickly, well, hard |
| 时间 | 何时 | yesterday, now, soon |
| 地点 | 在哪 | here, there, everywhere |
| 频率 | 多频繁 | always, often, never |
| 程度 | 多少 | very, quite, too |

### 底层理解

副词 = **给动作或描述加「周边信息」**。

### 标签

**副词（Adverb）** — 缩写 adv.

---

## 9. 介词（Preposition）

### 要解决的问题

「两个东西之间的**关系**是什么？」—— 位置、时间、方向、原因、方式……

### 它在句子里干什么

**不能单独作谓语**，必须跟名词/代词组成 **介词短语**，在句中通常作定语或状语。

| 关系 | 例子 |
|------|------|
| 位置 | **in** the room |
| 时间 | **at** 5 pm / **on** Monday |
| 方向 | **to** school |
| 方式 | **with** a knife |
| 原因/目的 | **for** money |

### 底层理解

介词 = **关系词**：英语很多关系不靠词尾变化，靠 **介词 + 名词** 这一小块。

### 标签

**介词（Preposition）** — 缩写 prep.

---

## 10. 连词（Conjunction）

### 要解决的问题

「**两个或多个想法**，之间是什么关系？」

### 它在句子里干什么

**连接**词、短语或从句。

| 类型 | 解决什么问题 | 例子 |
|------|-------------|------|
| **并列连词** | 平等地连两个想法 | and, but, or → Tom **and** Jerry |
| **从属连词** | 标明主从关系 | because, if, when → I stayed **because** it rained. |

### 底层理解

连词 = **想法之间的胶水**：并列、转折、因果、条件、时间……

### 标签

**连词（Conjunction）** — 缩写 conj.

---

## 11. 冠词（Article）

### 要解决的问题

「这个名词是**新信息**还是**双方都知道的那个**？可数还是不可数？」

### 它在句子里干什么

放在名词前，作 **限定词** 的一种。

| 冠词 | 解决什么问题 | 例子 |
|------|-------------|------|
| **a / an** | 第一次提，泛指一个 | **a** dog |
| **the** | 特指，双方都知道 | **the** dog |
| **零冠词** | 泛指一类 / 不可数 | I like **coffee**. |

### 底层理解

冠词 = **给名词标「新/旧、特指/泛指」**。

### 标签

**冠词（Article）** — 属于 **限定词（Determiner）** 的一种。

---

## 12. 限定词（Determiner）

### 要解决的问题

「说的是**哪一个 / 多少 / 哪些**？」—— 比冠词范围更广。

### 它在句子里干什么

放在名词前，限定名词范围。

| 类型 | 例子 |
|------|------|
| 冠词 | a, an, the |
| 指示 | this, that, these, those |
| 物主 | my, your, his |
| 数量 | some, any, many, few, all |
| 疑问 | which, what |

### 底层理解

限定词 = **名词前的「定位器」**：哪一个、多少个、谁的。

### 标签

**限定词（Determiner）** — 缩写 det.

---

## 13. 数词（Numeral / Number）

### 要解决的问题

「**多少 / 第几**？」

### 它在句子里干什么

| 类型 | 例子 |
|------|------|
| 基数词 | **three** apples |
| 序数词 | the **first** day |

### 底层理解

数词 = **给数量或顺序命名**，常作定语或主语。

### 标签

**数词（Numeral）** — 有时归入限定词或形容词。

---

## 14. 感叹词（Interjection）

### 要解决的问题

「**突然的情绪 / 反应**」—— 不属于句子主干。

### 它在句子里干什么

独立出现，表达惊讶、喜悦、疼痛等。

- **Oh!** / **Wow!** / **Ouch!** / **Hey!**

### 底层理解

感叹词 = **情绪的即时出口**，不参与句子主干结构。

### 标签

**感叹词（Interjection）** — 缩写 interj.

---

## 15. 非谓语动词：词的形式，不是单独的词类

> 这是 **动词的变化形式**，在句子里可以当名词、形容词、状语用。  
> 底层上：**同一个动作，用不同形式承担不同角色。**

### 15.1 不定式（to do）

#### 要解决的问题（两层）

**第一层：为啥要有「不定式」这种形式？**

一句话里 **谓语位置** 通常只能有一个真正的「主句动词」。但人还要说：

- **To learn** English is fun. → 「学英语」这件事当主语，不能再写 is learn
- I want **to go**. → 「去」是 want 的对象，不是整句谓语

**问题：** 怎么在 **不当主句谓语** 的情况下，还说「做某事」？

**解法：** 动词变成 **非谓语形式** —— **不定式 to do**（把动作打包成「一件事」）。

---

**第二层：为啥不定式前面要加 to？**

`to` 不是随便粘上去的，它解决 **两个问题**：

| 问题 | to 怎么解决 |
|------|------------|
| **1. 怎么标出「这是不定式，不是谓语」？** | to + 原形 = 标记「这个动词不当主句谓语用」 |
| **2. 怎么表示「朝向 / 针对某个动作」？** | to 本是介词「朝、向」（go **to** school），引申为 **针对某个将做或将说的动作** |

```
I want to go
        ↑  ↑
        │  动作本身（go）
        └── 朝向/针对「去」这件事
```

**底层理解：**  
不定式 = **把动作当「一件事」来说**；  
**to** = **路标**：这是 packaged 的动作，且常带有 **方向/目的/针对** 感（want **to** go = 想要 **朝着「去」** 那个方向）。

---

#### 不定式 vs 谓语动词

| | 谓语动词 | 不定式 to do |
|--|---------|-------------|
| 角色 | 主句核心：怎么了 | 打包的动作，不当主句核心 |
| 例子 | I **go**. | I want **to go**. |
| 人称/数 | 随主语变（go/goes/went） | **不变**（永远是 to go） |

**为啥叫「不定式」？**  
= **不随主语「定」变化** 的动词形式（对比：I go / he goes 会「定」）。

---

#### to do 在句子里干什么（句子成分）

| 成分 | 例子 | 解决什么问题 |
|------|------|-------------|
| **主语** | **To learn** is hard. | 把「做某事」当主语 |
| **宾语** | I want **to go**. | want 的对象是「去」这件事 |
| **宾补** | Ask him **to come**. | 请他 **来做** 某事（见 [底层英语.md 5.7](./底层英语.md)） |
| **定语** | I have work **to do**. | 要 **去做** 的工作 |
| **状语** | I came **to help**. | 来的 **目的是** 帮忙 |

---

#### 为啥有些是 to do，有些是不加 to 的 do？

**不是 leave 要不要 to，是主句动词属于哪一类框：**

| 框 | 结构 | 主句在干什么 | 动词 |
|----|------|-------------|------|
| **A** | sb **to do** | 对「将发生的事」表态 | allow, want, ask, tell |
| **B** | sb **do** | 直接放行/造成/感知 | let, make, see, hear |

详见 → [底层英语.md · 两套宾补框研究专题](./底层英语.md)

**诚实补充：** 部分原因是英语 **日耳曼老框**（无 to）和 **后来 to 不定式框** 并存。

---

#### 和动名词 -ing 比：为啥有时 to do，有时 doing？

| 形式 | 底层感觉 | 例子 |
|------|---------|------|
| **to do** | **要去做 / 将要做** 的那件事 | I want **to swim**.（想去游） |
| **-ing** | **做这件事** 本身（更当名词） | I enjoy **swimming**.（游泳这活动） |

---

#### 标签

**不定式（Infinitive）** = **to + 动词原形**  
**to** = 不定式标记 + 「朝向/针对该动作」的残留义

### 15.2 动名词（-ing，作名词用）

| 步骤 | 内容 |
|------|------|
| **问题** | 要把「做某事这件事」当作名词 |
| **解法** | 动词 + -ing |
| **例子** | **Swimming** is fun. / I enjoy **reading**. |
| **可充当** | 主语、宾语 |

### 15.3 分词（-ing / -ed）

| 形式 | 问题 | 例子 | 可充当 |
|------|------|------|--------|
| **现在分词** -ing | 主动 / 进行 | the **sleeping** baby | 定语、状语、补语 |
| **过去分词** -ed | 被动 / 完成 | a **broken** window | 定语、状语、补语 |

**底层理解：** 非谓语 = **动词不当谓语用时**，变形去干名词、定语、状语的活。

## 16. 词类 vs 句子成分：别混

| 维度 | 词类 | 句子成分 |
|------|------|---------|
| 问的是 | 这个**词本身**是什么类型 | 这个**词/短语在句中**干什么 |
| 例子 | happy 是**形容词** | happy 在 She is happy 里是**表语** |
| 关系 | 一个词类 → 可在句中充当多种成分 | 一个成分 → 可由多种词类来充当 |

**对照表：**

| 词类 | 常充当的句子成分 |
|------|----------------|
| 名词 / 代词 | 主语、宾语、表语 |
| 动词 | 谓语 |
| 形容词 | 定语、表语 |
| 副词 | 状语 |
| 介词短语 | 定语、状语 |
| 不定式 / 动名词 / 从句 | 看位置，多种成分都可能 |

句子成分详见 → [底层英语.md 第 5 节](./底层英语.md#5-句子成分全集每个零件解决什么问题)

## 17. 总对照：要表达什么 → 用什么词类

| 你想表达的信息 | 用什么词类 | 例子 |
|---------------|-----------|------|
| 谁/什么（命名） | 名词 | dog, idea |
| 谁/什么（不重复） | 代词 | he, it, someone |
| 怎么了/做了什么 | 动词 | run, eat, is |
| 时态/进行/被动/疑问/否定 | 助动词 | am, have, do |
| 能力/可能/必须/应该 | 情态动词 | can, must, should |
| 哪一种/什么样的 | 形容词 | red, happy |
| 怎样/何时/在哪 | 副词 | quickly, yesterday |
| 两物/两想法的关系 | 介词 / 连词 | in, because |
| 新/旧、特指/泛指 | 冠词 / 限定词 | a, the, my, some |
| 多少/第几 | 数词 | three, first |
| 情绪反应 | 感叹词 | Oh! Wow! |
| 动作当名词/修饰语 | 非谓语形式 | to do, -ing, -ed |

## 19. 一句话总结

**词类不是分类游戏，而是：**

> 你想传递某种信息 → 需要某种「角色」来承载 → 英语里这类角色通常由某类词来干 → 语法书管这类词叫「名词/动词/……」。

**实词说内容，虚词搭结构；先想表达什么，再认词类名字。**
