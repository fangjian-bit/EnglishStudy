# 底层英语

> **学这个，不是背语法名字。**  
> 是学：你想说一句话 → 必须回答哪些问题 → 英语用什么结构补上 → 语法书给这个结构起了什么名（最后才记）。  
> 词类（名词、动词、助动词等）见 → [词类.md](./词类.md)  
> **全部专题索引**见 → [索引.md](./索引.md)

---

## 怎么用这份笔记

每学一个「零件」，只问三个问题：

1. **没有它，会表达不了什么？**
2. **它解决的是哪个沟通问题？**
3. **能不能用自己的话造一个例子？**

三个都能答 → 懂了，标签顺便记一下。  
答不上来 → 别背标签，回到「你想说什么」。

**推荐顺序（永远这样走）：**

```
我想说什么（中文意图）
    ↓
这句话要回答哪些问题？
    ↓
英语用什么词/结构补上？
    ↓
语法书管这个叫什么？（标签，放最后）
```

**不要反过来：** 先背「谓语 = 动词」→ 遇到句子硬套 → 懵了。

## 1. 人说话，最少要回答哪些问题？

一句话不是随机拼词，是在回答一组问题：

| 你想让听者知道什么 | 中文例子 |
|-------------------|---------|
| 说的是**谁 / 什么**？ | **他**…… |
| **怎么了 / 做了什么**？ | ……**吃**…… |
| 动作**落到谁 / 什么**上？ | ……吃**苹果** |
| 是**什么 / 怎么样**（不是说动作）？ | 她**很开心** |
| **哪一种 / 多少个**？ | **红**色的车、**三**个 |
| **怎样 / 何时 / 在哪**？ | **很快地**、**昨天**、**在学校** |
| 是**新信息**还是**双方都知道的**？ | **一个**人 vs **那个**人 |
| 发生在**什么时候**？跟现在什么关系？ | 昨天做了 / 正在做 / 将要做 |
| 是**事实**还是**可能 / 应该 / 必须**？ | 他可能来 / 你必须走 |
| 有**两个想法**，什么关系？ | 因为下雨，所以没出门 |

英文句子的零件，就是围绕这些问题长出来的。**先记住问题，再记零件。**

## 2. 实操：从中文意图到英文结构

别从标签入手，从**你想说什么**入手。

### 例 1：「他吃了一个苹果」

| 步骤 | 内容 |
|------|------|
| 意图 | 说一个人吃东西 |
| 要回答的问题 | 谁？→ 他。怎么了？→ 吃。吃啥？→ 苹果 |
| 英文 | He ate an apple. |
| 语法标签（最后才看） | 主语 He / 动词 ate / 宾语 an apple → **主谓宾** |

### 例 2：「她很开心」

| 步骤 | 内容 |
|------|------|
| 意图 | 说一个人的状态，不是说她在「做」什么事 |
| 要回答的问题 | 谁？→ 她。怎么样？→ 开心 |
| 英文 | She is happy. |
| 语法标签 | 主语 She / 系动词 is / 表语 happy → **主系表**（不是主谓宾） |

### 例 3：「狗在叫」

| 步骤 | 内容 |
|------|------|
| 意图 | 说一个动作，不需要额外对象 |
| 要回答的问题 | 谁？→ 狗。怎么了？→ 叫 |
| 英文 | The dog barks. |
| 语法标签 | 主语 The dog / 动词 barks → **主谓** |

### 例 4：「你喜欢这个吗？」「喜欢。」

| 步骤 | 内容 |
|------|------|
| 意图 | 回答喜不喜欢 |
| 要回答的问题 | 谁？→ 我。怎么了？→ 喜欢。（喜欢什么？→ 问句里已经有了） |
| 英文 | I like. / I do. / Yes. |
| 底层逻辑 | **不是 like 不需要对象，是对话里对象已知，省略了**（见第 6 节） |

### 例 5：「他们选他当主席」

| 步骤 | 内容 |
|------|------|
| 意图 | 说选举结果：他变成了主席 |
| 要回答的问题 | 谁？→ 他们。怎么了？→ 选。选谁？→ 他。选完什么结果？→ 主席 |
| 英文 | They elected him president. |
| 语法标签 | 主语 They / 谓语 elected / 宾语 him / **宾补** president |

### 例 6：「我昨天在公园见到的那只大狗」

| 步骤 | 内容 |
|------|------|
| 意图 | 不是随便哪条狗，是「昨天在公园见到的、大的」那条 |
| 要回答的问题 | 哪一个？→ 昨天在公园见到的；什么样的？→ 大的 |
| 英文 | the big dog **that I saw in the park yesterday** |
| 语法标签 | big = **定语**（形容词）；that I saw… = **定语从句** |

**练法：** 以后每学一句，先在脑子里走这四步，再去看标签对不对。

## 3. 句子最小要有什么？

### 要解决的问题

只听一个名字（She.），没有下文 → 听者不知道你想说什么。  
所以最少要有：

1. **锚点**：说的是谁 / 什么
2. **说明**：这个锚点怎么了 / 是什么 / 怎么样

```
She  |  is happy
锚点     说明
The dog  |  barks
锚点        说明
```

### 英语解法

| 沟通问题 | 英语解法 | 语法书叫 |
|---------|---------|---------|
| 说的是谁/什么 | 句首那一块 | **主语** |
| 怎么了/是什么/怎么样 | 动词（及后面必要的补充） | **谓语**（= 动词或系动词） |

**最小完整句 = 主语 + 谓语（动词）。** 宾语、表语不是每句都要，是动词意思不够用时才补上的（下一节）。

### 关于「谓语」——只记一种，别混

学 **主谓宾 / 主系表** 时：

> **谓语 = 动词（或系动词）。** 就这一个词（或词组核心）。

```
He    ate    an apple
主语   谓语     宾语

She   is     happy
主语   谓语     表语
```

- `an apple` 是 **宾语**，不是谓语
- `happy` 是 **表语**，不是谓语
- 只有 `ate` / `is` / `barks` 这些，才是谓语

> 注：有些英文教材把「主语以外的全部」也叫 Predicate（谓语），范围更大。  
> **本笔记统一用主谓宾体系：谓语 = 动词。** 别两套混着记。

## 4. 动词说了算：后面还要不要加东西？

**核心逻辑：不是句子规定动词后面有什么，是动词的意思决定还缺什么信息。**

### 情况 A：动词自己就说完了

**问题：** 动作 / 状态不需要对象。

| 中文 | 英文 | 还缺什么 |
|------|------|---------|
| 我睡觉 | I sleep. | 不缺 |
| 狗叫 | The dog barks. | 不缺 |
| 他跑了 | He runs. | 不缺 |

→ 语法标签：**主谓**。动词叫 **不及物动词**（意思不指向外部对象）。

### 情况 B：动词意思里带着「对象」

**问题：** 只说动词，听者会问「什么？」

| 中文 | 动词 | 听者会问 | 要补什么 | 英文 |
|------|------|---------|---------|------|
| 我吃 | 吃 | 吃啥？ | 苹果 | I eat an apple. |
| 我喜欢 | 喜欢 | 喜欢啥？ | 咖啡 | I like coffee. |
| 他知道 | 知道 | 知道啥/谁？ | 真相 | He knows the truth. |

→ 语法标签：**主谓宾**。后面补的叫 **宾语**。动词叫 **及物动词**（意思指向对象）。

### 情况 C：不是在说「做了什么」，是说「是什么 / 怎么样」

**问题：** 不是在描述动作，是在描述主语本身。

| 中文 | 英文 | 缺什么 | 补什么 |
|------|------|-------|-------|
| 她很开心 | She is happy. | 怎么样？ | happy |
| 他是老师 | He is a teacher. | 是什么？ | a teacher |
| 汤很香 | The soup smells good. | 怎么样？ | good |

→ 动词（is / smells）几乎没动作，作用是 **连接主语和描述**，叫 **系动词**。  
→ 后面描述主语的那块叫 **表语**（不是宾语——宾语是动作的对象，表语是主语自身的说明）。  
→ 语法标签：**主系表**。

### 一张表收束

| 你想表达什么 | 动词特点 | 后面补什么 | 结构 | 例子 |
|-------------|---------|-----------|------|------|
| 做了某事（不需对象） | 不及物 | 无 | 主谓 | I sleep. |
| 做了某事（有对象） | 及物 | 宾语 | 主谓宾 | I eat an apple. |
| 是什么 / 怎么样 | 系动词 | 表语 | 主系表 | She is happy. |

**兼类动词**（eat / read / sing 等）：两种说法都行，看你想不想提对象 → `I eat.` / `I eat rice.`  
常见词见 → [单词积累.md](./单词积累.md)

## 5. 句子成分全集：每个零件解决什么问题？

> 语法书列的「句子成分」，不是要你背 9 个名字，而是：**每种成分对应一个你要回答的问题。**  
> 下面逐个走：**意图 → 问题 → 解法 → 标签**。

### 总览

| 成分 | 它在回答什么问题 | 一句话 |
|------|----------------|--------|
| **主语** | 说的是谁/什么？ | 句子的锚点 |
| **谓语** | 怎么了/做了什么/是什么状态？ | 动词核心 |
| **宾语** | 动作落到谁/什么上？ | 动作的承受者 |
| **表语** | 主语是什么/怎么样？ | 描述主语本身 |
| **定语** | 是哪一个/什么样的？ | 修饰名词 |
| **状语** | 怎样/何时/何地/为何？ | 修饰动作或整句 |
| **宾补** | 动作之后对象变成啥样？ | 补充说明宾语 |
| **同位语** | 进一步解释前面那个名词是谁/是什么？ | 名词的重述 |
| **从句** | 某一块信息太复杂，一个词说不完 | 嵌套的一整句 |

---

### 5.1 主语

| 步骤 | 内容 |
|------|------|
| **意图** | 「我想说的这件事，主体是谁/什么」 |
| **问题** | 说的是谁/什么？ |
| **解法** | 放句首（或 it/there 占位），作锚点 |
| **标签** | **主语** |

**例句：**

- `The dog` barks. → 狗在叫
- `Swimming` is fun. → 游泳这件事很有趣
- `What he said` is true. → 他说的那句话是真的

**主语可以是什么形态：**

| 你想说的锚点 | 英语解法 | 例子 | 标签 |
|-------------|---------|------|------|
| 具体的人/物 | 名词 | The **dog** barks. | 名词作主语 |
| 已经知道是谁 | 代词 | **He** is happy. | 代词作主语 |
| 一整件事/行为 | -ing | **Swimming** is fun. | 动名词作主语 |
| 「去做某事」这件事 | to do | **To learn** is hard. | 不定式作主语 |
| 一整句想法 | 从句 | **What he said** is true. | 主语从句 |
| 主语太长 | it 占位 | **It** is fun **to swim**. | 形式主语 |
| 某处有什么 | there + 名词 | **There** is a book. | 存在句 |
| 两个一起 | A and B | **Tom and Jerry** are friends. | 并列主语 |

**判断法：** 问「是谁/什么是 happy / fun / true？」→ 答案就是主语。

常见词见 → [单词积累.md](./单词积累.md)

---

### 5.2 谓语

| 步骤 | 内容 |
|------|------|
| **意图** | 「这个主体，怎么回事」 |
| **问题** | 怎么了 / 做了什么 / 是什么状态？ |
| **解法** | 用动词（或系动词）说明 |
| **标签** | **谓语**（= 动词 / 系动词） |

**例句：**

- He **ate** an apple.
- She **is** happy.
- The dog **barks**.

> 本笔记统一：**谓语 = 动词**。宾语、表语、状语等是动词周围的其他成分，不叫谓语。

---

### 5.3 宾语

| 步骤 | 内容 |
|------|------|
| **意图** | 「动作作用到了谁/什么上」 |
| **问题** | 吃啥？给谁？看见谁？ |
| **解法** | 放在及物动词后面 |
| **标签** | **宾语** |

**例句：**

- I eat **an apple**. → 吃 → 苹果
- I like **coffee**. → 喜欢 → 咖啡
- He gave me **a book**. → 给 → 书

**两种常见宾语：**

| 类型 | 回答的问题 | 例子 | 标签 |
|------|-----------|------|------|
| **直接宾语** | 动作直接作用到什么 | I read **a book**. | 直接宾语 |
| **间接宾语** | 动作涉及谁（常指人） | He gave **me** a book. | 间接宾语 |

`He gave me a book.` = 他给了一本书给我 → me（给谁）= 间接宾语，a book（给什么）= 直接宾语。

**介词宾语：** 有些动词不直接接宾语，要加介词 → `listen to music` / `wait for me`  
→ music / me 是 **介词的宾语**，不是动词的直接宾语。

**宾语从句：** 宾语是一整句想法 → `I know **that he is right**.`

---

### 5.4 表语

| 步骤 | 内容 |
|------|------|
| **意图** | 「不是在说做了什么，是在说主语本身是什么/怎么样」 |
| **问题** | 她怎么样？他是什么？ |
| **解法** | 系动词 + 描述 |
| **标签** | **表语**（主系表结构） |

**例句：**

- She is **happy**. → 她怎么样？→ 开心
- He is **a teacher**. → 他是什么？→ 老师
- The soup smells **good**. → 汤怎么样？→ 香

**表语 vs 宾语（别混）：**

| | 表语 | 宾语 |
|--|------|------|
| 前面动词 | 系动词（is/look/feel） | 动作动词（eat/like/give） |
| 说明谁 | **主语本身** | **动作的承受对象** |
| 例子 | She is **happy** | She likes **coffee** |

**表语从句：** 表语是一整句 → `The problem is **that we have no time**.`

---

### 5.5 定语

| 步骤 | 内容 |
|------|------|
| **意图** | 「不是随便哪个，是**这一种/这一个**」 |
| **问题** | 哪一种？什么样的？哪个的？ |
| **解法** | 放在名词前面或后面，修饰名词 |
| **标签** | **定语** |

**例句：**

| 中文意图 | 英文 | 定语是什么 | 形式 |
|---------|------|-----------|------|
| 红色的车 | a **red** car | red | 形容词 |
| 桌上的书 | the book **on the desk** | on the desk | 介词短语 |
| 我昨天买的手机 | the phone **I bought yesterday** | I bought yesterday | 定语从句 |
| 正在睡觉的婴儿 | the **sleeping** baby | sleeping | 现在分词 |

**判断法：** 问「哪个/什么样的 + 名词？」→ 修饰名词的那块就是定语。

---

### 5.6 状语

| 步骤 | 内容 |
|------|------|
| **意图** | 「动作/整句话的周边信息：怎样、何时、在哪、为啥」 |
| **问题** | 怎样？何时？在哪？为什么？ |
| **解法** | 修饰动词、形容词、副词或整句 |
| **标签** | **状语** |

**例句：**

| 回答的问题 | 中文 | 英文 | 状语形式 |
|-----------|------|------|---------|
| 怎样？ | 很快地跑 | He runs **quickly**. | 副词 |
| 何时？ | 昨天离开 | **Yesterday**, I left. | 副词 |
| 在哪？ | 在学校 | at school | 介词短语 |
| 为什么？ | 因为下雨 | **because** it rained | 状语从句 |
| 如果… | 如果你来 | **If** you come, I'll go. | 状语从句 |

**定语 vs 状语（别混）：**

| | 定语 | 状语 |
|--|------|------|
| 修饰谁 | **名词** | **动词 / 形容词 / 整句** |
| 回答 | 哪一个/什么样的 | 怎样/何时/在哪/为何 |
| 例子 | a **red** car | runs **quickly** |

---

### 5.7 宾语补足语（宾补）

| 步骤 | 内容 |
|------|------|
| **意图** | 「动作做完之后，对象处于什么状态/变成什么」 |
| **问题** | 使/让/发现/选 … 之后，它怎么样了？ |
| **解法** | 宾语后面再补一块说明 |
| **标签** | **宾语补足语**（宾补） |

**结构：动词 + 宾语 + 宾补**

```
keep    the room    clean
elect   him         president
call    him         Tom
        ↑ 宾语       ↑ 宾补（动作之后对象的状态/身份）
```

**底层逻辑：** 动词意思里带着「对象 + 结果/状态」，所以宾语后面还要补一块。

> 中文常加「当/作/为/成」：选他**当**总统。英文 often 直接：elect him president。

---

#### 宾补是什么词性？

**宾补是句子成分，不是单一词性**——形容词、名词、动词原形、介词短语、分词、不定式等 **都可以** 当宾补。

---

#### ① 形容词作宾补（最常见）

**问题：** 动作之后，宾语 **怎么样？**

| 中文 | 英文 | 宾语 | 宾补 | 词性 |
|------|------|------|------|------|
| 保持房间整洁 | Keep the room **clean**. | the room | clean | **形容词** |
| 使我开心 | Make me **happy**. | me | happy | **形容词** |
| 我觉得这很有趣 | I find it **interesting**. | it | interesting | **形容词** |
| 把门打开 | Leave the door **open**. | the door | open | **形容词** |
| 把头发弄湿 | Get your hair **wet**. | your hair | wet | **形容词** |
| 把价格定低 | Keep the price **low**. | the price | low | **形容词** |
| 使大家安静 | Keep everyone **quiet**. | everyone | quiet | **形容词** |
| 把墙刷白 | Paint the wall **white**. | the wall | white | **形容词** |

**心里换算：** the room **is** clean / he **became** happy

---

#### ② 名词作宾补

**问题：** 动作之后，宾语 **是什么身份/名字？**

| 中文 | 英文 | 宾语 | 宾补 | 词性 |
|------|------|------|------|------|
| 选他当总统/主席 | They elected him **president**. | him | president | **名词** |
| 叫他 Tom | We call him **Tom**. | him | Tom | **名词** |
| 任命她经理 | They appointed her **manager**. | her | manager | **名词** |
| 选他为队长 | We chose him **captain**. | him | captain | **名词** |
| 尊称他为老师 | We call him **teacher**. | him | teacher | **名词** |
| 命名这狗 Lucky | They named the dog **Lucky**. | the dog | Lucky | **名词** |
| 认定他为专家 | We consider him **an expert**. | him | an expert | **名词短语** |

> `elect him president` = 选他 **（当）** 总统。英文可写 elect him **as** president，更正式。

**心里换算：** he **became** president / he **is called** Tom

---

#### ③ 动词原形作宾补（使役/让某人做）

**问题：** 动作之后，宾语 **去做什么**？

| 中文 | 英文 | 宾语 | 宾补 | 词性 |
|------|------|------|------|------|
| 让我走 | Let me **go**. | me | go | **动词原形** |
| 让他试 | Let him **try**. | him | try | **动词原形** |
| 使我去 | Make me **go**. | me | go | **动词原形** |
| 让狗跑 | Make the dog **run**. | the dog | run | **动词原形** |
| 帮我开一下 | Help me **open** it. | me | open | **动词原形** |
| 我看见他跑 | I saw him **run**. | him | run | **动词原形** |
| 听见她唱 | I heard her **sing**. | her | sing | **动词原形** |

> 使役动词 make/let + 宾语 + **动词原形**（不加 to）。详见下方 [研究专题：两套宾补框](#研究专题两套宾补框--为啥-to-do为啥不加-to)。

#### ④ 不定式作宾补（to do）

**问题：** 动作之后，宾语 **要去做什么**？

| 中文 | 英文 | 宾语 | 宾补 | 词性 |
|------|------|------|------|------|
| 请他来 | Ask him **to come**. | him | to come | **不定式** |
| 想要我帮忙 | Want me **to help**. | me | to help | **不定式** |
| 允许他们走 | Allow them **to leave**. | them | to leave | **不定式** |
| 叫她去 | Tell her **to go**. | her | to go | **不定式** |
| 期望他成功 | Expect him **to succeed**. | him | to succeed | **不定式** |
| 促使我们思考 | Lead us **to think**. | us | to think | **不定式** |

> get + 宾语 + **to do**：I got him **to help**.（让他说服去做了）

#### 研究专题：两套宾补框 —— 为啥 to do，为啥不加 to？

> 见 [③ 动词原形](#-动词原形作宾补使役让某人做) 与 [④ 不定式](#-不定式作宾补to-do) 的对比。  
> **不是背「哪个动词加 to」，而是研究：主句动词在解决什么沟通问题。**

##### 先纠正一个误区

| 句子 | 说话时走了吗 |
|------|-------------|
| Allow them **to leave**. | ❌ 都可以 **还没走** |
| Let them **leave**. | ❌ 都可以 **还没走** |

**所以：不是因为「走了没走」决定加不加 to。**  
差别在 **英语用两种「框」组织「许可/放行 + 离开」。**

---

##### 框 A：动词 + 宾语 + **to do**（带 to）

```
allow + them + [ to leave ]
              ↑
        「被允许的那件事的内容」
```

| 步骤 | 内容 |
|------|------|
| **主句在干什么** | 对 **「某件将发生的事」** 表态（许可、愿望、请求、预期） |
| **to 解决什么问题** | 标出 **态度所指向的那件事** —— 许可的是「离开」这个内容 |
| **to 的古义感** | 朝向、针对（和 go to school 的 to 同类） |
| **代表动词** | allow, want, ask, tell, expect, need（人作主语时） |

**心里换算：** 我允许 **（他们离开这件事）** → Allow them **to leave**.

**自检：** 主句 **不是** 离开本身，而是 **对「离开」的态度** → 要 **to do**。

---

##### 框 B：动词 + 宾语 + **do**（动词原形，无 to）

```
let + them + [ leave ]
            ↑
      直接接动作，不再套 to 短语
```

| 步骤 | 内容 |
|------|------|
| **主句在干什么** | **直接放行、直接造成、直接感知** 这个动作 |
| **为啥无 to** | 动作 **捆在** 致使/感知框里，不是「对一件事的内容表态」 |
| **代表动词** | let, make, see, hear, watch, feel |

**心里换算：** 我 **不拦了，他们走** → Let them **leave**（不是「批准一份内容」，是「放行」）。

**自检：** ~~let that they leave~~ ❌ —— let **不接「一件事的内容」**，只接「人 + 做」。

---

##### Allow vs Let 对照（同一「还没走」）

| | Allow them **to leave** | Let them **leave** |
|--|-------------------------|-------------------|
| **中文感** | **准许**「离开」**这件事** | **不拦了**，**他们走** |
| **主句解决什么** | 对某事 **给许可** | **不再阻止**，让动作发生 |
| **结构** | 框 A：态度 + **to** + 内容 | 框 B：放行 + **直接接动作** |
| **场景感** | 规定、制度：You're **allowed to leave** after 3. | 当场：OK, I'll **let you leave** now. |
| **Let's leave** | — | = Let us leave，**咱们走吧**，也还没走，也无 to |

---

##### 为啥英语有两套框？（历史，诚实说）

现代英语 ≈ **日耳曼语老习惯** + **法语/拉丁语新习惯** 并存：

| 来源 | 接动作的方式 | 例子 |
|------|-------------|------|
| 日耳曼老传统 | 宾语 + **动词原形** | let, make, see, hear |
| 后来发展 | 宾语 + **to do** | allow, want, ask, expect |

**allow** 从法语进入英语，走「对一件事表态 → to do」；  
**let** 是日耳曼老动词，历史上就是 **let + 人 + 做**，不带 to。

**研究结论：** 今天不能 100% 用一条规则从中文推出每个动词，但能 **按主句动词分派** 理解，不是死背单词。

---

##### 动词分派：看主句在干什么

| 主句在干什么 | 结构 | 动词举例 |
|-------------|------|---------|
| 对 **将发生的事** 表态（许可/愿望/请求/预期） | sb **to do** | allow, want, ask, tell, expect |
| **直接** 放行、迫使、看见 | sb **do** | let, make, see, hear |
| 两种都可（少数） | help me (**to**) carry | help |

---

##### 错句 vs 对句

| ❌ 错 | ✅ 对 | 为啥 |
|------|------|------|
| Allow them leave | Allow them **to leave** | allow 走框 A，要 to 标「被允许的内容」 |
| Let them to leave | Let them leave | let 走框 B，不加 to |
| Make him to go | Make him go | make 走框 B |
| Want him go | Want him **to go** | want 走框 A |
| I saw him to leave | I saw him leave | see 走框 B（感知动作，不是指向将来） |

---

##### 一条研究用规则

```
主句是在对「某件将发生的事」表态？
    → 是 → sb + to do（allow / want / ask …）

主句是直接放行、迫使、或亲眼看见？
    → 是 → sb + do 原形（let / make / see …）
```

**一句话：to 不是给 leave 加的，是框 A 用来标「态度所指向的那件事」；let 走框 B，直接接 leave，所以无 to—— 且两种都可以是「还没走」。**

> **不定式 to do 为啥存在、to 本身干什么？** → 见 [词类.md · 15.1 不定式](./词类.md)

---

#### ⑤ 现在分词作宾补（-ing，正在做）

**问题：** 动作发生时，宾语 **正在做什么**？

| 中文 | 英文 | 宾语 | 宾补 | 词性 |
|------|------|------|------|------|
| 看见他在跑 | I saw him **running**. | him | running | **现在分词** |
| 听见她在哭 | I heard her **crying**. | her | crying | **现在分词** |
| 发现他们在笑 | I found them **laughing**. | them | laughing | **现在分词** |
| 保持机器运转 | Keep the machine **running**. | the machine | running | **现在分词** |

**对比：**
- I saw him **run**. → 看见跑（全过程或做了跑这个动作）
- I saw him **running**. → 看见 **正在** 跑

#### ⑥ 过去分词作宾补（-ed，被动/完成状态）

**问题：** 动作之后，宾语 **处于被…的状态**？

| 中文 | 英文 | 宾语 | 宾补 | 词性 |
|------|------|------|------|------|
| 想要它修好的 | I want it **fixed**. | it | fixed | **过去分词** |
| 需要工作做完 | I need the work **done**. | the work | done | **过去分词** |
| 让问题被解决 | Get the problem **solved**. | the problem | solved | **过去分词** |
| 保持联系畅通 | Keep us **connected**. | us | connected | **过去分词** |

#### ⑦ 介词短语作宾补

**问题：** 动作之后，宾语 **在什么位置/状态**？

| 中文 | 英文 | 宾语 | 宾补 | 词性 |
|------|------|------|------|------|
| 把书放桌上 | Put the book **on the desk**. | the book | on the desk | **介词短语** |
| 请让大家出去 | Keep everyone **outside**. | everyone | outside | **副词/介词短语** |
| 放它在这里 | Leave it **here**. | it | here | **副词** |

> 有的语法书把「地点/方向」归入 **状语** 而非宾补，两种分析都有。**底层上：** 都是补充说明动作之后宾语的位置/状态。

#### 常跟宾补的动词（按类型记）

| 类型 | 常见动词 | 宾补常是什么词性 |
|------|---------|----------------|
| 使/让/保持 | make, let, have, keep, leave, get | 形容词、动词原形、分词 |
| 认为/发现 | find, consider, think, believe | 形容词、名词、to do |
| 称呼/选举 | call, name, elect, appoint, choose | **名词** |
| 感官 | see, hear, watch, feel | 动词原形、现在分词、过去分词 |
| 要求/允许 | ask, tell, want, allow, expect | **不定式 to do** |
| 放置 | put, place, leave | 介词短语、形容词 |

#### 宾补 vs 表语（别混）

| | 宾补 | 表语 |
|--|------|------|
| 说明谁 | **宾语** | **主语** |
| 例子 | They elected **him** president. | **He** is president. |
| 例子 | I find **it** interesting. | **It** is interesting. |

#### 判断法

1. 找 **动词 + 宾语**
2. 问：**动作之后，宾语是什么/怎么样/在做什么？**
3. 后面那一块 = **宾补**
4. 再看宾补本身是形容词、名词还是 to do / -ing 等


### 5.8 同位语

| 步骤 | 内容 |
|------|------|
| **意图** | 「前面那个名词，我再解释一下，就是谁/是什么」 |
| **问题** | 这个人/这个东西，具体是谁/是什么？ |
| **解法** | 放在名词后面，重述/解释同一个东西 |
| **标签** | **同位语** |

**例句：**

- My friend **Tom** came. → Tom = My friend（同一个人）
- The fact **that he lied** surprised me. → that he lied 解释 fact 的内容
- Beijing, **the capital of China**, is large. → the capital of China 解释 Beijing

**同位语 vs 定语从句（别混）：**

| | 同位语 | 定语从句 |
|--|--------|---------|
| 作用 | **解释/重述**名词 | **限定/修饰**名词 |
| 例子 | my friend **Tom** | the man **who called you** |

---

### 5.9 从句：当一个成分太长时

**要解决的问题：** 主语、宾语、表语、定语、状语……某一块信息 **一个词说不完**，就嵌一整句。

| 从句当什么成分 | 回答的问题 | 例子 |
|--------------|-----------|------|
| **主语从句** | 说的是谁/什么（一整句） | **What he said** is true. |
| **宾语从句** | 动作涉及什么（一整句） | I know **that he is right**. |
| **表语从句** | 主语是什么（一整句） | The problem is **that we have no time**. |
| **定语从句** | 哪一个/什么样的（一整句） | The man **who called you** is my boss. |
| **状语从句** | 何时/为何/如果…（一整句） | I stayed **because** it rained. |

**判断法：** 先找句子主干（谁 + 怎么了），剩下的从句看它在修饰/补充谁 → 对号入座。

---

### 5.10 独立成分（知道即可）

不是句子的主干，插进来表达态度、补充说明、呼叫等：

| 类型 | 解决什么问题 | 例子 | 标签 |
|------|-------------|------|------|
| **插入语** | 顺便插一句补充/态度 | **Frankly**, I disagree. | 插入语 |
| **呼语** | 直接叫某人 | **Tom**, come here! | 呼语 |
| **感叹词** | 表达情绪 | **Oh**, I see! | 感叹词 |

---

### 5.11 一句里怎么找成分？（实操）

以 `The big dog quickly ran to the park yesterday.` 为例：

| 步骤 | 问什么 | 答案 | 成分 |
|------|--------|------|------|
| 1 | 谁/什么？ | The big dog | **主语**（big 是定语，修饰 dog） |
| 2 | 怎么了？ | ran | **谓语** |
| 3 | 怎样？ | quickly | **状语** |
| 4 | 去哪？ | to the park | **状语** |
| 5 | 何时？ | yesterday | **状语** |

**口诀：先找主干（主 + 谓 + 宾/表），再挂修饰（定语、状语），最后看有没有宾补、同位语、从句。**

## 6. 对话里，已知的信息可以不说

### 要解决的问题

对话里重复已知信息，说话累赘。中文英文都一样。

| 问 | 答 | 省略了什么 | 为啥能说 |
|----|-----|-----------|---------|
| 你喜欢这个吗？ | 喜欢。 | 「这个」 | 问句里有了 |
| Do you like it? | I like. / I do. / Yes. | it / like it | 同上 |

### 两层逻辑（别混）

| 层级 | 说什么 |
|------|--------|
| **动词本身** | like 意思里带着「喜欢什么」→ 及物动词 |
| **对话里** | 「喜欢什么」双方都知道 → 省略宾语 |

**不是语法改了，是上下文替你把缺的信息补上了。** 语法书管这个叫 **省略（ellipsis）**。

孤立说 `I like.` → 像没说完。  
别人刚问 `Do you like it?` → 完全正常。

## 7. 其他常见零件（冠词、时态、连词等）

> 定语、状语、宾补等同属 **句子成分**，见 [第 5 节](#5-句子成分全集每个零件解决什么问题)。  
> 下面是 **不属于主干成分、但日常必用** 的零件。

### 冠词 a / the —— 「新的还是双方都知道的？」

- **a** dog → 第一次提，随便一只
- **the** dog → 特指，双方都知道
- 标签：**冠词**（常作名词前的定语）

### 时态 —— 「什么时候的事？跟现在什么关系？」

- I **work** → 习惯 / 一般事实
- I **worked** → 已发生
- I **will work** → 还没发生

### 进行 / 完成 —— 「进行到哪了？」

- I **am working** → 正在进行
- I **have worked** → 跟现在有联系

### 情态动词 —— 「事实 / 可能 / 应该 / 必须？」

- You **must** go. / You **might** go.

### 被动语态 —— 「强调动作还是承受者？」

- Someone broke the window. → 强调谁干的
- The window **was broken**. → 强调窗碎了

### 连词 —— 两个想法什么关系？

- I stayed **because** it rained. → 因果（引导状语从句）
- **If** you come, I'll go. → 条件

## 8. 语序：先听骨架，再听细节

**问题：** 听者怎么最快听懂？

英文习惯：**谁 + 做了什么** 放前面，修饰放后面。

```
The big dog quickly ran to the park yesterday.
     ↓
主干：dog ran
     ↓
再叠：大的、快、去公园、昨天
```

中文更灵活；英文 **先骨架，后装饰**。

## 9. 总对照：问题 → 解法 → 标签

| 你想解决什么问题 | 英语怎么补 | 语法书叫 |
|-----------------|-----------|---------|
| 说的是谁/什么 | 句首锚点 | **主语** |
| 怎么了/做了什么 | 动词 | **谓语** |
| 动作落到谁/什么 | 动词后补对象 | **宾语** |
| 给/告诉涉及谁 | 动词后补人 | **间接宾语** |
| 主语是什么/怎么样 | 系动词 + 描述 | **表语** |
| 是哪一个/什么样的（名词） | 名词前/后修饰 | **定语** |
| 怎样/何时/在哪/为何 | 动词/句首/句尾修饰 | **状语** |
| 动作后对象什么状态 | 宾语后再补 | **宾补** |
| 解释前面那个名词 | 名词后重述 | **同位语** |
| 某成分太复杂 | 嵌一整句 | **从句**（主/宾/表/定/状语从句） |
| 新信息还是已知 | a / the | **冠词** |
| 什么时候的事 | 动词变形 | **时态** |
| 可能/应该/必须 | can/must/should | **情态动词** |
| 强调承受者 | be + 过去分词 | **被动语态** |
| 两个想法的关系 | and/but/because | **连词** |
| 已知信息不说 | 省略重复部分 | **省略** |

## 10. 语法标签速查（附录：实在需要查名字时再看）

> 标签是 **地图**，不是 **目的地**。  
> 先会从意图造句，再回来对号入座。

| 标签 | 它在回答什么问题 | 典型例子 |
|------|----------------|---------|
| 主语 | 谁/什么 | **He** ate an apple. |
| 谓语 | 怎么了/做了什么 | He **ate** … / She **is** … |
| 宾语 | 动作落到谁/什么 | He ate **an apple**. |
| 间接宾语 | 给/告诉谁 | He gave **me** a book. |
| 表语 | 主语是什么/怎么样 | She is **happy**. |
| 定语 | 哪一个/什么样的 | a **red** car / the book **on the desk** |
| 状语 | 怎样/何时/在哪/为何 | runs **quickly** / **Yesterday** / **because** it rained |
| 宾补 | 动作后对象什么状态 | They elected him **president**. |
| 同位语 | 解释前面那个名词 | my friend **Tom** |
| 主语从句 | 主语是一整句 | **What he said** is true. |
| 宾语从句 | 宾语是一整句 | I know **that he is right**. |
| 定语从句 | 修饰名词的一整句 | the man **who called you** |
| 状语从句 | 时间/原因/条件等 | I stayed **because** it rained. |
| 主谓 | 动作不需对象 | I **sleep**. |
| 主谓宾 | 动作有对象 | I **eat** rice. |
| 主系表 | 描述主语 | She **is** a teacher. |
| 及物动词 | 意思指向对象 | like, eat, want |
| 不及物动词 | 意思不指向对象 | sleep, arrive, happen |
| 省略 | 上下文已知则不说 | Do you like it? — **Yes.** |

## 11. 一句话总结

**英语语法不是一堆要背的名字，而是：**

> 你想说一句话 → 必须回答哪些问题 → 英语用哪些词补上 → 语法书给这些词起了名字。

**研究底层，研究的是「问题 → 解法」，不是「标签 → 例句」。**
